# wm

外贸 SaaS ERP 原型（进销存 + 报关 + 单证）。

## 本次优化（基于外贸行业场景深度增强）

围绕外贸企业常见链路（采购 → 销售 → 库存 → 报关 → 单证），已做如下优化：

1. 信息模型增强（基础资料）
- 商品增加外贸关键字段：
  - SKU、HS Code、报关品名、材质、原产国
  - 装箱参数（每箱数、长宽高）
  - 单件毛重/净重、安全库存
- 新增客户主数据：国家、币种、默认贸易条款（Incoterm）、默认港口
- 供应商增加国家与币种

2. 业务功能扩展（多做功能）
- 登录与角色权限（admin / purchase / sales / finance）
- 采购入库（含供应商、条款、ETA、币种、港口）
- 销售出库（含客户联动、条款、港口、币种）
- 库存查询（含低库存预警）
- 报关单模块（按销售自动汇总 HS、毛净重、金额）
- 单证中心（商业发票 + 装箱单汇总）
- 导入导出中心（CSV 模板、导入、库存/报关导出）
- 经营看板（采购/销售/毛利/低库存）

3. 页面交互样式统一
- 每个业务页面都改成 Excel 风格：**表头 + 表体**
- 表体按行录入，支持下拉联动

## SPA 路由

- `/login` 登录
- `/base-data` 基础资料
- `/purchase` 采购入库
- `/sales` 销售出库
- `/inventory` 库存查询
- `/customs` 报关单
- `/documents` 单证中心
- `/io-tools` 导入导出
- `/analytics` 经营看板

## 运行方式

```bash
cd /workspace/wm
python3 -m http.server 5500 -d frontend
```

浏览器访问：

- `http://127.0.0.1:5500/spa.html`

## 技术栈

- Vue 3 + Vue Router + Element Plus（CDN）
- 前端内存状态管理（reactive store）
